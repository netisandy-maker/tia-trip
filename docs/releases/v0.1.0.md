# TIA Trip v0.1.0

Initial public release of TIA Trip.

## Highlights

- Expense splitter web UI with trip/member/expense workflow
- ASP.NET Core API endpoints for trips, members, expenses, and summary
- Settlement calculation logic
- Playwright tests in TypeScript
- Playwright tests in C# (.NET)
- CI workflow for TS and C# suites
- Fixture layer for TS Playwright tests
- BDD-style test suite and Gherkin scenarios
- Wiki content for onboarding and interview prep

## Included paths

- `src/api/TiaTrip.Api`
- `src/web/index.html`
- `tests/e2e-ts`
- `tests/e2e-dotnet/TiaTrip.E2E.Dotnet`
- `tests/bdd`
- `.github/workflows/ci.yml`
- `docs/wiki`

## Notes

- Local run:
  - `npm run api:start`
  - `npm run web:start`
- Test run:
  - `npm run test:ts`
  - `npm run test:bdd`
  - `dotnet test tests/e2e-dotnet/TiaTrip.E2E.Dotnet/TiaTrip.E2E.Dotnet.csproj`
