# How to Publish This Wiki to GitHub

GitHub Wiki uses its own repository. The easiest path:

1. Open your repo on GitHub.
2. Click **Wiki**.
3. Click **Create the first page**.
4. Copy content from `docs/wiki/Home.md` into the page titled `Home`.
5. Create another page named `Learning-Map` and paste `docs/wiki/Learning-Map.md`.

Optional (advanced): clone `<repo>.wiki.git` and push markdown files directly.
